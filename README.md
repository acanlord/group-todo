# TodoList Full Stack App 
To-Do-ish is a minimalistic web application that save tasks, displays a calendar, and provides today's weather.

# Install npm
npm install 

# Start Back End
./start_backend.sh

# Start Front End
./run.sh

# Features
Add new task<br/>
Delete a task<br/>
Choose a date from the calendar<br/>
Search today's weather

# Technologies Used
HTML<br/>
CSS<br/>
JavaScript<br/>
MongoDB

# TodoList Full Stack App 
To-Do-Ish is a minimalistic web application that save tasks, displays a calendar, and provides today's weather.

# Install npm
npm install 

# Start Back End
./start_backend.sh

# Start Front End
./run.sh

# Features
Add new task
Delete a task
Choose a date from the calendar
Search today's weather

# Technologies Used
HTML
CSS
JavaScript
MongoDB
